# SEO Implementation Plan (oli-theme — Cycle 1, Plan 7/10)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development.

**Goal:** Build the full custom SEO module : per-content meta storage, full `<head>` composition (title, description, canonical, hreflang, robots, Open Graph, Twitter Card, JSON-LD `@graph` multi-schemas), multilingual XML sitemap, breadcrumbs builder, readability score (Flesch-FR Kandel-Moles), keyword analyzer, internal link suggester, image auditor, 301/410 redirects with admin UI, per-post SEO metabox with live counters + SERP preview + score gauge, SEO Overview admin dashboard. Match or exceed Yoast on the features that matter for olikalari.com (multilingual native, JSON-LD `@graph` complete, French readability validated, image audit, internal linking).

**Architecture:** Mirror of previous modules — DTOs + models + controllers + module. JSON-LD schemas implemented via `SchemaInterface` (each schema is a single-responsibility class returning a JSON-serializable array). `SchemaContext` aggregates them under `@graph`. The `<head>` builder runs at `wp_head` priority 1 and outputs everything in one shot. Score is configurable via `apply_filters('oli_seo_score_rules', $rules)` so each site can tune weights.

**Tech Stack:** Same as Plans 1-6.

**Reference spec:** `docs/superpowers/specs/2026-05-05-oli-theme-design.md` — section 4.1 (Module SEO complet).

### T20 — `ReadabilityAnalyzer` (Flesch-FR Kandel-Moles)

Class with `score(string $text): int` (0-100, plus haut = plus lisible). Implémente la formule :

```
Flesch_FR = 207 - 1.015 * (mots/phrases) - 73.6 * (syllabes/mots)
```

(Kandel & Moles 1958 pour le français, vs 84.6 pour Flesch original anglais).

Helpers privés : `countSentences(string)`, `countWords(string)`, `countSyllables(string)`.

`countSyllables` : approximation par regex sur les voyelles (ae, ie, eu = 1 syllabe ; chaque cluster séparé = 1 ; règles spéciales pour `e` final muet, `qu`, etc.). Acceptable de viser une précision approchée (±5%) — c'est plus qu'assez pour un score lisibilité.

Tests : 4 (`testEmptyTextReturnsZero`, `testSimpleFrenchSentenceScoresHigh`, `testComplexLegalTextScoresLow`, `testCountSyllablesIsApproximate`).

Commit: `feat(seo): ajoute ReadabilityAnalyzer (Flesch-FR Kandel-Moles)`.
