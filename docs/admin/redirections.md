# Redirections

`Admin → Apparence du thème → SEO → Redirections` : gérer les redirections 301 personnalisées.

## Ajouter une redirection

1. Renseigner l'**URL source** (chemin relatif, ex. `/ancienne-page/`).
2. Renseigner l'**URL cible** (chemin relatif ou URL absolue).
3. Enregistrer.

## Quand utiliser une redirection

- Une page a été renommée et l'ancien lien circule encore (réseaux, autres sites).
- Migration depuis l'ancien site (les anciens slugs renvoient vers les nouveaux).
- Article supprimé : rediriger vers la catégorie ou la home, plutôt qu'une 404.

## 301 vs 302

Le thème applique du **301 permanent** (recommandé pour le SEO). Si la cible est provisoire, prévenir le développeur pour adapter.
