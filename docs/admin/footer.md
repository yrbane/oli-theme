# Pied de page (footer)

## Réseaux sociaux

`Admin → Apparence du thème → Contact → Réseaux sociaux` : renseigner les URLs des comptes (Facebook, Instagram, etc.). Les icônes apparaissent automatiquement en pied de page, avec un **hover aux couleurs de marque**.

## Évolutions prévues

Le footer va être enrichi avec :

- Un **logo dédié** au footer (sélecteur médiathèque).
- Un **texte libre** (mentions légales, signature) éditable depuis l'admin.

Voir [#13](https://github.com/yrbane/oli-theme/issues/13) pour le suivi.
