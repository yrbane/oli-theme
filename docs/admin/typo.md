# Typographie

> **À venir** : un panneau de réglage des tailles de typographie est planifié — voir l'issue [#11](https://github.com/yrbane/oli-theme/issues/11).

## En attendant

Les tailles sont définies par la variation CSS active :

- **Titres** : utilisent la police choisie dans `Apparence → Police des titres`, avec une échelle modulaire.
- **Texte courant** : ~1 rem (16 px par défaut), avec une largeur de lecture optimisée à ~65 caractères par ligne.
- **Menu** : ~1 rem sur desktop, réduit sur tablette/mobile.

## Si le menu déborde sur deux lignes

Voir le guide **Menus** — c'est souvent dû à la longueur des titres et non à la typographie.
