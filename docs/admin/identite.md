# Identité visuelle

L'onglet **Identité & Marque** regroupe les éléments d'identité du site.

## Logo, favicon, médias

- **Logo principal** : visible dans l'en-tête et le pied de page. Utilise le sélecteur de médiathèque WordPress (bouton « Choisir une image »).
- **Favicon** : icône affichée dans l'onglet du navigateur. Format carré recommandé (au moins 512 × 512 px).
- **Slogan** : phrase courte affichée sous le titre du site (paramétrable indépendamment du « tagline » WordPress).

## Bonnes pratiques

- Privilégier le format **SVG** pour le logo (mise à l'échelle parfaite).
- Pour les images bitmap, utiliser des fichiers **WebP** ou **PNG** optimisés.
- Toujours renseigner un **texte alternatif** (champ « texte de remplacement » dans la médiathèque) — essentiel pour l'accessibilité et le SEO.
