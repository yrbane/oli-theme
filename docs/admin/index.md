# Aide du thème oli-theme

Bienvenue Olivier. Cette page rassemble toute la documentation nécessaire pour utiliser le thème, sans avoir à sortir de l'admin WordPress.

## Sommaire des guides

### Identité & contenu
- **Identité visuelle** : logo, favicon, slogan, médias.
- **Bannière responsive** : comment l'image s'adapte aux 3 supports (desktop/tablette/mobile).
- **Apparence & variations** : variations CSS, police des titres, couleurs.
- **Typographie** : régler la taille de base, l'échelle des titres, le menu, le footer.
- **Contenu (articles & pages)** : cartes d'article home avec vignettes, page d'accueil par langue.
- **Slides du carousel** : créer une slide, choisir son image mise en avant.
- **Menus** : construire le menu, comportement responsive (burger à 992 px), longueur des titres.
- **Langues & traductions** : activer FR/EN, lier les versions, audit des contenus à traduire.
- **Pied de page** : logo, texte libre, mentions légales par langue.

### Médias & galeries
- **Galerie photos & vidéos** : page Photos agrégée, page Vidéos YouTube, lightbox.
- **Dossiers médiathèque** : organiser les images par dossiers + sous-dossiers, bulk actions, upload auto dans un dossier, shortcode `[oli_folder_gallery]`.

### Présentation visuelle avancée
- **Gabarits — styles de présentation** : 12 styles interchangeables (parallaxe, brutalist, cinéma, magazine, triptyque, story…) + architecture zonale (zones texte/image/galerie remplies par contenu).

### Réseaux sociaux & sync
- **Réseaux sociaux** *(onglet « Comptes » sous Réseaux sociaux)* : URLs des comptes affichés en footer.
- **Synchro Facebook + Instagram** : OAuth Meta, publication automatique au publish/update/delete, gestion des erreurs.

### Réservations
- **Calendrier de réservation** : planning hebdomadaire admin, services CRUD, widget de réservation frontend, notifications FR/EN, sync iCal (Gmail/Apple/Outlook).

### SEO & maintenance
- **SEO** : tableau de bord, scores, métadonnées par contenu.
- **Redirections** : 301 personnalisées entre URLs.

## Astuce

Chaque champ de réglage du thème porte une petite bulle **?** : cliquer dessus ouvre directement le guide correspondant.

## Auto-documentation : où trouver les guides

- **Cet onglet** (`Apparence du thème → Aide & documentation`) regroupe tout.
- Les guides sont des fichiers Markdown versionnés sous `docs/admin/*.md` — édités en code, pas en base de données. La doc évolue avec le thème.
