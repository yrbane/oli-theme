# Aide du thème oli-theme

Bienvenue Olivier. Cette page rassemble toute la documentation nécessaire pour utiliser le thème, sans avoir à sortir de l'admin WordPress.

## Sommaire des guides

- **Identité visuelle** : logo, favicon, slogan, médias.
- **Bannière responsive** : comment l'image s'adapte aux 3 supports.
- **Apparence & variations** : variations CSS, police des titres, couleurs.
- **Typographie** : régler la taille des titres et des textes.
- **Contenu** : articles, pages, page d'accueil.
- **Slides du carousel** : créer une slide et choisir son image.
- **Galerie photos & vidéos** : ajouter et organiser les médias.
- **Menus** : construire et adapter le menu principal.
- **Langues & traductions** : activer les langues, lier les versions FR/EN.
- **Pied de page** : réseaux sociaux, logo et mentions.
- **SEO** : tableau de bord, scores, métadonnées.
- **Redirections** : redirections 301 personnalisées.
- **Réseaux sociaux** : comptes affichés en footer.

## Astuce

Chaque champ de réglage du thème porte une petite bulle **?** : cliquer dessus ouvre directement le guide correspondant.
