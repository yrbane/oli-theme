# SEO

`Admin → Apparence du thème → SEO → Tableau de bord` : vue d'ensemble du SEO de chaque contenu.

## Lecture du tableau

- **Score SEO** : note synthétique (titre, meta description, longueur, structure Hn, alt des images).
- **Filtres** : par type de contenu, par score, par langue.
- **Export CSV** : bouton dédié pour analyse externe.

## Métadonnées par article/page

Chaque éditeur d'article expose une **metabox SEO** :

- Titre SEO (balise `<title>`).
- Meta description (snippet Google).
- Image partage social (OpenGraph).

## Bonnes pratiques

- Titre : 50–60 caractères.
- Meta description : 140–160 caractères, factuelle, avec verbe d'action.
- Image OpenGraph : 1200 × 630 px.
